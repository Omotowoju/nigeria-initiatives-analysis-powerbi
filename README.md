# Nigeria Initiatives Analysis (Power BI)

## Overview
This project analyzes initiative programs across Nigeria, focusing on Agriculture, Medical, and STEM sectors.

## Tools Used
- Power BI
- Data Cleaning
- Data Visualization

## Key Insights
- Identified regions with the highest concentration of initiatives
- Highlighted gaps in program distribution
- Compared sector performance across locations

## Outcome
This dashboard provides insights that can support better decision-making and resource allocation.
